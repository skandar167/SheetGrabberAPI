Présentation de SheetGrabber

SheetGrabber est un outil web qui permet de géocoder inversement des coordonnées GPS (latitude / longitude) contenues dans un fichier Excel, afin d'en extraire automatiquement des informations de localisation : commune, ville, état, code postal, adresse complète, etc.

📂
Import Excel

Supporte les formats .xlsx et .xls avec glisser-déposer.

🌍
Géocodage automatique

Conversion de coordonnées en adresse complète via API.

🗺️
Carte interactive

Visualisation des clients sur une carte plein écran avec leur nom.

⬇️
Export Excel

Téléchargement du fichier enrichi avec toutes les données de localisation.

2

Préparer votre fichier Excel

Pour utiliser SheetGrabber, votre fichier doit contenir au minimum les colonnes suivantes :

📌 Colonnes requises

 • latitude ou lat ou y — Latitude GPS du client
 • longitude ou lng / lon ou x — Longitude GPS
 • Toute colonne contenant le nom complet du client (ex: nom, name, client)

✅
Bonne pratique : Nommez vos colonnes avec des en-têtes clairs en première ligne. SheetGrabber détecte automatiquement les colonnes de latitude et longitude.
⚠️
Attention : Les coordonnées doivent être des nombres décimaux valides (ex: 36.7372, 3.0869). Les valeurs vides, nulles ou égales à 0 seront ignorées.
3

Importer le fichier Excel

  1. Ouvrez SheetGrabber dans votre navigateur.
  2. Dans la zone "Import Excel File", cliquez sur la zone d'upload ou glissez-déposez votre fichier .xlsx / .xls directement.
  3. Le nom du fichier et sa taille s'affichent en confirmation.
  4. Un aperçu des 10 premières lignes apparaît automatiquement dans la section "Data Preview".
ℹ️
Note : Seule la première feuille du fichier Excel est traitée. Si vos données sont dans une autre feuille, déplacez-les dans la première avant d'importer.
4

Configurer les colonnes

Une fois le fichier importé, la section "Coordinate Mapping" apparaît. Vous devez associer les colonnes de votre fichier aux champs requis :

🔧 Trois colonnes à sélectionner

 • Latitude Column — La colonne contenant les latitudes GPS
 • Longitude Column — La colonne contenant les longitudes GPS
 • Client Name Column — La colonne avec le nom complet du client (pour la carte)

🤖
Détection automatique : SheetGrabber reconnaît automatiquement les colonnes standard. Si vos colonnes sont nommées lat, lng, nom ou similaire, elles seront pré-sélectionnées.
5

Lancer le géocodage

Une fois les colonnes configurées, cliquez sur le bouton "Start Geocoding".

  1. Le traitement démarre ligne par ligne, avec une barre de progression en temps réel.
  2. Chaque ligne est traitée à raison d'une requête par seconde (limite de l'API).
  3. Le journal d'état indique la ligne en cours de traitement.
  4. À la fin, les statistiques affichent le nombre de succès, d'erreurs et de lignes ignorées.
⏱️
Durée estimée : Comptez environ 1 seconde par ligne. Pour 100 clients : ~1 minute 40. Pour 500 clients : ~8 minutes. Ne fermez pas la page pendant le traitement.
6

Visualiser les clients sur la carte

Après avoir sélectionné les colonnes latitude, longitude et nom, la section "Visualisation Carte" apparaît dans la colonne de droite.

  1. Cliquez sur "Ouvrir la carte en plein écran".
  2. La page carte s'ouvre et affiche tous vos clients comme des marqueurs sur la carte.
  3. La sidebar gauche liste tous les clients avec leur nom.
  4. Utilisez la barre de recherche pour trouver un client par nom.
  5. Cliquez sur un client dans la liste pour que la carte vole vers sa position.
  6. Cliquez sur un marqueur sur la carte pour voir le nom complet du client.
  7. Cliquez sur "◀ Retour" pour revenir à la page principale.
💡
La carte est utilisable avant même le géocodage, dès que vous avez sélectionné les colonnes latitude, longitude et nom.
7

Exporter les résultats

Après la fin du géocodage, la section "Export Processed Sheet" apparaît :

  1. Cochez les colonnes originales de votre fichier à conserver.
  2. Cochez les nouvelles colonnes géocodées à inclure dans l'export.
  3. Cliquez sur "Download Processed Excel File".
  4. Un fichier processed_addresses_AAAAMMJJ.xlsx est téléchargé automatiquement.
📦 Colonnes ajoutées par SheetGrabber

commune   municipality   town   district   suburb   city   state   country   postcode   full_address   geocoding_status

8

Comprendre les statuts de géocodage

La colonne geocoding_status indique le résultat du traitement pour chaque ligne :

✅ success ❌ error ⏭ skipped
✅ success

La coordonnée a été correctement géocodée. Toutes les colonnes d'adresse sont remplies.

❌ error

Une erreur s'est produite lors de la requête API (erreur réseau, serveur indisponible, etc.). Relancez le traitement si ce cas se produit fréquemment.

⏭ skipped

La ligne a été ignorée car les coordonnées sont vides, nulles (0,0) ou au format invalide (NaN, texte, etc.).

9

Conseils et bonnes pratiques

📱
Numéros de téléphone : Si votre fichier contient des colonnes nommées tél, tel, portable ou mobile, les numéros commençant par 0 sont automatiquement convertis au format international algérien +213XXXXXXXXX dans l'export.
💾
Sauvegardez pendant le traitement : Ne rafraîchissez pas la page en cours de géocodage, cela réinitialisera toutes les données. Attendez la fin complète avant d'exporter.
🌐
Connexion Internet requise : SheetGrabber nécessite une connexion active pour appeler l'API de géocodage. L'import, la carte et l'export fonctionnent en local.
📊
Grands fichiers : Pour les fichiers de plus de 500 lignes, prévoyez suffisamment de temps. La barre de progression vous indiquera l'avancement en temps réel.
